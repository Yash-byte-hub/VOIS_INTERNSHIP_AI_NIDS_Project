# AI-Based Network Intrusion Detection System

This project demonstrates an AI-powered Network Intrusion Detection System (NIDS) using Machine Learning.

## Features
- Random Forest based intrusion detection
- Simulated network traffic data
- Real-time dashboard using Streamlit
- Classification of traffic as Benign or Malicious

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- Seaborn, Matplotlib

## How to Run
1. Create a virtual environment
2. Install dependencies
3. Run:
   streamlit run nids_main.py

This project is developed for academic and cybersecurity awareness purposes.
